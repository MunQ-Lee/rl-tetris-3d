# 3D Tetris — Competitive Reinforcement Learning (PPO vs PPO)

두 개의 강화학습(PPO) 에이전트가 **3차원 테트리스로 대전**하며 학습합니다.
한쪽이 여러 줄(레이어)을 동시에 지우면 **상대 보드 바닥에 가비지 블록이 쌓여
올라가고**, 천장을 넘은 쪽이 패배합니다. 두 모델의 대결을 웹 브라우저(three.js)
에서 실시간 3D로 볼 수 있습니다.

- **State**: 보드의 형태 특징(높이·구멍·표면 거칠기 등) + 현재/다음 블록
- **Action**: **배치(placement)** — 현재 블록을 놓을 수 있는 모든 최종 위치
  `(회전, x, y)` 중 하나를 골라 바로 하드드롭 (1스텝 = 1조각).
  Tetris RL의 정석인 *아터스테이트* 방식이라 학습이 실제로 됩니다.
- **Reward**: 레이어 클리어 점수 + 보드 품질(구멍·높이 감소) + 공격(가비지 전송)
  보너스 − 피격 − 승/패 보너스
- **경쟁**: 두 에이전트가 서로를 상대로 self-play 하며 같이 강해집니다.

```
보드: 6×6×12,  조각: 8종 tetracube
가비지 전송: 2줄→1, 3줄→2, 4줄→4 (한 번에 여러 줄을 지워야 공격)
```

## 구성

| 파일 | 설명 |
|------|------|
| `tetris3d/pieces.py`          | 조각 정의 · 24방향 회전 · 배치 드롭 기하 |
| `tetris3d/env.py`             | 배치형 3D 테트리스 (벡터화 아터스테이트, 보상, 커리큘럼, 가비지) |
| `tetris3d/afterstate_agent.py`| 마스킹 PPO (배치 점수 정책 + value net) |
| `tetris3d/versus.py`          | 2인 대전 환경 (동시 배치 · 가비지 교환 · 승패) |
| `train.py`                    | 두 에이전트 경쟁 학습 루프 (커리큘럼 decay) |
| `server.py` + `web/index.html`| 두 보드 나란히 대전 장면 실시간 GUI |
| `push_if_improved.py`         | 성능이 오르면 체크포인트를 GitHub에 push |

## 설치

```bash
pip install -r requirements.txt
```

## 1) 학습 (두 모델 경쟁)

```bash
python train.py --updates 5000 --rollout 1024
```

- `checkpoints/latest_p0.pt`, `latest_p1.pt` (GUI용), `best_p0.pt` / `best_p1.pt` 저장.
- 초반에는 **커리큘럼**(거의 다 찬 보드에서 시작)으로 첫 클리어를 쉽게 배우고,
  점차 빈 보드 실전으로 넘어갑니다.
- 죽었다 재시작해도 `latest_p*.pt`에서 **이어서 학습**합니다.

## 2) 대전 장면 보기 (GUI)

```bash
python server.py --port 8000 --speed 6
```

브라우저에서 **http://localhost:8000** 접속 — 왼쪽 PLAYER 0(파랑), 오른쪽
PLAYER 1(빨강) 보드가 동시에 플레이됩니다. 한쪽이 줄을 지우면 상대 보드가
회색 가비지로 차오르는 것을 볼 수 있습니다. 체크포인트를 자동 핫리로드하므로
학습이 진행되며 두 모델이 강해지는 과정을 실시간으로 확인할 수 있습니다.

## 작동 방식 — 왜 배치(아터스테이트)인가

매 스텝 블록을 한 칸씩 이동/회전시키는 방식은, 희소한 보상으로 내비게이션과
클리어 전략을 동시에 배워야 해서 사실상 학습이 안 됩니다(실제로 5×5/6×6에서
100만+ 스텝 동안 클리어 0이었습니다). 그래서 **가능한 모든 최종 배치를 나열하고
그 결과 보드(아터스테이트)의 품질로 한 번에 고르는** 방식으로 바꿨습니다. 보상도
모든 배치가 뚜렷이 차등화되도록 보드 품질(구멍·높이) 기반으로 강하게 설계해,
정책이 따라갈 그래디언트가 생깁니다.

대전에서는 여기에 가비지 공격/피격과 승패 보너스가 더해져, 두 모델이 단순 생존을
넘어 **여러 줄을 모아 지워 공격하는** 전략까지 경쟁적으로 학습합니다.
